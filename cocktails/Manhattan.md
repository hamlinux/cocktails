### Description


### Ingredients:
| Ingredients       | Amount |
| ----------------- | ------ |
| Rye Whiskey       | 1.5    |
| Sweet Vermouth    | .75    |
| Angostura Bitters | 3 dash |
|                   |        |

### Garnish:
3 Cocktail Cherries

### Directions:
-  Combine ingredients over ice in mixing glass. 
- Stir over ice for 20-30 seconds.
- Single Strain into chilled coupe.
- Garnish.
