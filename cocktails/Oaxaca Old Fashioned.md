### Description

### Ingredients:
| Ingredients       | Amount   |
| ----------------- | -------- |
| Reposado Tequila  | 1.25     |
| Mezcal            | .50      |
| Demerra Syrup     | .25      |
| Angostura Bitters | 3 Dashes |

### Garnish:
Flamed Orange Twist


### Directions:
-  Combine ingredients over ice in mixing glass. 
- Stir  20-30 seconds.
- Single Strain into chilled rocks glass over large ice cube.
- Garnish
